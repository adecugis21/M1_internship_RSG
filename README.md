**Overview**

This repository accompanies the M1 internship research project “Deciphering the Sequence-Encoded Grammar of Chromatin Organization with Cross-Species Multi-Modal Deep Learning”.
It contains the code, scripts, and environment files used for model training, dataset-specific analyses, and figure generation.
The project explores the use of deep learning architectures to decode chromatin structural features encoded in DNA sequences, integrating cross-species genomic and epigenomic data.
